Daftar Pemrograman Web

HTML
HTML adalah singkatan dari Hypertext Markup Language adalah bahasa yang perlu anda pelajari terlebih dahulu, sebelum mempelajari pemrograman lainnya, karena seluruh web saat ini menggunakan HTML sebagai dasarnya
CSS
CSS adalah singkatan dari Cascading Style Sheets merupakan bahasa yang digunakan untuk melakukan penataan layout dari website
Javascript
Javascript adalah bahasa pemrograman yang digunakan untuk membuat tampilan web anda lebih menarik dengan menggunakan effect - effect khusus
PHP
PHP adalah bahasa pemrograman web yang digunakan untuk mengatur logika pemrograman dalam sebuah web, dan mengatur sebuah pengiriman dan penerimaan data dari database
